# Gestionnaire de mots de passe - Flask

## ✅ Lancement local
1. `pip install -r requirements.txt`
2. `python app.py`
3. Va sur http://localhost:10000

## ☁️ Hébergement gratuit sur Render
1. Crée un compte sur https://render.com
2. Nouveau Web Service :
   - Langage : Python
   - Start command : python app.py
   - Build command : pip install -r requirements.txt
   - Public
3. Déploie ce dossier en ZIP ou en repo Git
